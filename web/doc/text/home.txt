Core concepts
-------------
The framework consists of a number of core functionalities:

<ul>
  <li>[doc://core/classloading Class loading] - class naming, <tt>uses()</tt>, 
      packages, static intializers, anonymous class creation
  </li>
  <li>[doc://core/xars XAR archives] - creating, using, class loading integration,
      the <tt>xar</tt> command.
  </li>
  <li>[doc://core/types Type system] - the <tt>lang.Type</tt> class, primitives,
      classes, type wrappers, type hierarchy
  </li>
  <li>[doc://core/generics Generics] - parametrized types, <tt>create()</tt>,
      declaration, benefits
  </li>
  <li>[doc://core/reflection Reflection] - the <tt>lang.XPClass</tt> class,
      invoking methods, accessing fields, packages, types
  </li>
  <li>[doc://core/annotations Annotations] - meta-data for classes, fields
      and methods
  </li>
  <li>[doc://core/enum Type-safe enumerations] - the <tt>lang.Enum</tt> class,
      attaching functionality to enum members</li>
</ul>

Examples

This section will give an introduction to the basics of common tasks and

Api documentation
-----------------
<em>See also: The [state://api complete overview].</em>

<summary>
Core
~~~~
Reflection, types, class loading, date manipulation, properties, profiling, 
collections framework and logging. 

Packages: [package://lang lang], [package://util util]
</summary>

<summary>
I/O
~~~
Local and remote I/O, files, folders, archives, flatfile databases, sockets,
servers and various protocols implementation. 

Packages: [package://io io], [package://peer peer]
</summary>

<summary>
Databases
~~~~~~~~~
RDBMS access APIs, connection manager, reverse engineering, O/R mapping. 

Packages: [package://rdbms rdbms]
</summary>

<summary>
XML
~~~
DOM- and SAX parsing, XPath, XSLT, tree and node data xml creation, XML / 
object mapping. 

Packages: [package://xml xml]
</summary>

<summary>
Web
~~~
Http scriptlets, portlets API, XML / XSL based tier solution, SOAP, WDDX,
JSON and XMLRPC

Packages: [package://scriptlet scriptlet], [package://webservices webservices]
</summary>

<summary>
Imaging
~~~~~~~
Image creation and manipulation, graphs, EXIF and IPTC extraction support.

Packages: [package://img img]
</summary>

<summary>
Text
~~~~
Encoders and decoders, formatting, tokenizing and parsing. 

Packages: [package://text text]
</summary>

<summary>
Remote
~~~~~~
Remote method invocation on Application Servers.

Packages: [package://remote remote]
</summary>

<summary>
Test-driven development
~~~~~~~~~~~~~~~~~~~~~~~
Unittests.

Packages: [package://unittest unittest]
</summary>
