
Setting up the framework
========================
The releases are offered as <tt>tar.gz</tt> or <tt>zip</tt> files which 
contain lang.base.php and the xp runtime.

Prerequisites
-------------
You need to download and install the [http://de3.php.net/ PHP language] 
to use the XP framework. PHP version >= 5.2.0 is required. The following 
extensions are needed:
<ul>
  <li>[php://pcre pcre] -  Perl compatible regular expressions</li>
  <li>[php://tokenizer tokenizer] -  Tokenizes strings</li>
  <li>[php://reflection reflection] -  Reflection API</li>
  <li>[php://ctype ctype] -  Character type functions</li>
  <li>[php://iconv iconv] -  Encoding / decoding</li>
  <li>[php://date date] -  Date and time functionality</li>
  <li>[php://dom dom] -  Document Object Model</li>
  <li>[php://xsl xsl] -  Extensible Stylesheets</li>
  <li>[php://xml xml] -  XML parsing</li>
  <li>[php://libxml libxml] -  XML library internals</li>
  <li>[php://sockets sockets] -  Socket functions</li>
  <li>[php://session session] -  Sessioning</li>
  <li>[php://gd gd] -  Graphics backend</li>
  <li>[php://exif exif] -  EXIF support</li>
  <li>[php://ftp ftp] - File transfer protocol implementation</li>
</ul>
The database access APIs for the various database systems require the 
respective drivers. See [package://rdbms the rdbms package].

Step by step
------------
Follow these steps to install the framework on your workstation:

Preparation
~~~~~~~~~~~
Create a directory where you want the XP framework installed to. For 
example: <tt>mkdir ~/xp</tt>. Change to this directory.

Download
~~~~~~~~
Type <tt>wget http://xp-framework.net/downloads/releases/VERSION/setup -O - | php</tt>.
This installer will get the necessary files and place them in a subdirectory
named after the version.

Add to path
~~~~~~~~~~~
<summary>
<ul>
  <li>Un*x: Add the following line to the file <tt>.profile</tt> in your 
      home directory: <tt>export PATH=$PATH:/home/thekid/xp/5.6.7/bin</tt>.
      If the <tt>.profile</tt> file does not exist, create it.
  </li>
  <li>Windows: Press the Windows-Key and "Pause" simultaneously to open
      the "system properties" dialog window. On the "Extended" tab, press
      the button labled "Environment variables". Add a variable called 
      <tt>Path</tt> (or edit an existing one) in the section "user variables"
      and add <tt>C:\Users\Thekid\xp\5.6.7\bin</tt> as value.
  </li>
</ul>
</summary>

Done
~~~~
You're ready to use the XP framework now! To verify your setup, open a
*new* shell window (so the environment variables are re-read) and type
<tt>xp -v</tt>.


Using additional packages
-------------------------
Additional packages (Google Search implementation, Delicio.us, Flickr, 
Java Webstart, Dia.org UML classes, and many more) are provided in the 
lib directory and must be downloaded separately.

Download
~~~~~~~~
Download the xar files from the releases' lib directory into the same 
directory where <tt>xp-rt-X.X.X.xar</tt> resides.

Done
~~~~
The package is now available and can be used.
