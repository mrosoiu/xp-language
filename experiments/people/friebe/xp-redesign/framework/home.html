use::xp

The XP framework offers consistent and well-tested APIs, a solid platform for private and enterprise applications. It will run anywhere PHP runs, Windows, Linux, Unix, MacOS, on your desktop PC, on your enterprise application cluster and on your hoster's webserver.
It's completely free, to use, to copy, to extend, to modify and even to redistribute.

Features

  • Enterprise Application Server Connectivity
  • O/R-mapping API
  • Collections framework
  • XML/XSL scriptlets
  • Unittests
  • Web services: Client/Server

Favorites

  • RFCs

Software

Dialog is a photo blog software written in XP. It features offline resizing and sharpening of the selected images to save bandwidth and web server CPU time, it's skinnable, doesn't require a database and is easy to set up.

See it in action at the core developers' homepages (Timm, Alex) or check out the dialog ports page.
