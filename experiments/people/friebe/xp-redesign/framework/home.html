use::xp

The XP framework offers consistent, multi-purpose, object oriented, production-tested collection of classes, APIs for app server connectivity, web services, dynamic web sites, date access and manipulation, logging, collections, I/O, databases, O/R mapping, XML, unittesting, and much more.

Features

  • Enterprise Application Server Connectivity
  • O/R-mapping API
  • Collections framework
  • XML/XSL scriptlets
  • Unittests
  • Web services: Client/Server

Software

Dialog is a photo blog software written in XP. It features offline resizing and sharpening of the selected images to save bandwidth and web server CPU time, it's skinnable, doesn't require a database and is easy to set up.

See it in action at the core developers' homepages (Timm, Alex) or check out the dialog ports page.
